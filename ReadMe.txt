WARNING:
	This program is intended to be used with the ImageGlass program for viewing images. 
	It works with any program but has the best slide show effect in ImageGlass.

INSTRUCTIONS:
	Check the enabled box to enable the slideshow effect
	This will open a random image from the designated folder on a timer
	The time can be set to the left of the enabled box, the linked box will set the min and max boxes to always be the same.
	If they are unlinked you can set different min and max times and a random time within the range will be chosen each time an image is opened
